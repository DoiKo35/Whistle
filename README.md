## 🌟 Features

*   **Flexible Profile Customization**  
    Change your avatar, set a custom nickname, provide a phone number, and customize your "About me" bio in a convenient info window.
*   **Username Search**  
    No need to share personal data. Find other users and view their profiles using a unique username.
*   **Device Management**  
    Security first. Control active sessions in the "Devices" tab and remotely terminate old connections.
*   **System Notifications**  
    Receive instant notifications about new logins to your account from the system bot, *Whistle Notifications*.

---

## 🛠 Tech Stack

The client is built using modern web technologies and a desktop application framework:

*   **Frontend / Desktop:** HTML5, CSS3, JavaScript, Electron
*   **Backend:** Node.js, Express, WebSockets (`socket.io`)
*   **Database:** SQLite
